My Projects

3D Shop: The 3D Interactive Online Shopping Experience

Solo Work

Web Development, UX Research, User Testing

3D Shop is my capstone project, representing the culmination of my Human Computer Interaction studies at RIT. This online store page was built to imitate an Amazon product page, but also allows users to interact with a featured product in 3D for a more engaging and streamlined experience. This project was heavily rooted in research and development, ultimately concluding with an academic study in which participants tested the prototype and submitted evaluations.

Autism Resources Hub: Everything You Need to Know in One Place

Group Work

Wireframe Prototype, UX Design, User Research

The web currently has tons of resources relating to autism, but it can be difficult to know where to look. Autism Resources Hub aims to solve this problem by centralizing existing information into a community managed repository.
